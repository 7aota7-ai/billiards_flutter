# System Overview v2.1

## 目的

本システムは、ビリヤードの練習・試合・検証・プロ動画から得た情報を蓄積し、試合で自然に再現できる判断力へ変換するための競技力向上システムである。

単なる練習記録ではなく、

```text
観察 → 構造化 → 実験 → 知識化 → 試合で活用
```

のサイクルを継続し、競技力を向上させる。

---

## 現在フェーズ

現在は **GAS再構築フェーズ** である。

旧GASは複数ファイルに分散し、実シート名不一致・入力規則破壊・数式列上書き・Today's Focus履歴崩れのリスクがあるため、旧GASを延命せず、新規 `system_v2.gs` を作成する。

```text
旧GAS：削除しない・実行しない
新GAS：system_v2.gs
設計資料：現行Spreadsheet構造に合わせて更新
```

## シート間の役割

| シート | 役割 |
|---|---|
| 使い方 | 利用方法・運用メモ |
| 判断基準 | 判断ルールの参照 |
| マスタ | 入力候補・共通選択肢 |
| 練習記録_archive | 練習記録の基礎データ（v2.0でManual化・リネーム済み。新規記録は練習ログへ一本化） |
| メニュー設計 | 練習メニュー案 |
| 変化量マップ_archive | 旧実験ログ（v2.0でManual化・リネーム済み。新規検証はPosition Labへ一本化） |
| Position Lab | 90度分離を基準にした条件別の実験・検証 |
| Pro Reference | プロ動画から得た観察・学び |
| 練習ログ | 日々の練習実施ログ |
| Today's Focus履歴 | Today's Focusの履歴保存 |
| 試合振り返り | 試合単位の総括 |
| ミスログ | 1ショット・1判断単位の事象分析 |
| 課題管理 | 継続して改善する課題 |
| 決定事項 | 競技原則・判断基準 |
| リファレンス | 検証済みの確定知識 |
| Today's Focus | 当日表示用ダッシュボード |
| 集計 | 集計表示 |
| _集計補助 | 集計補助用 |

## 設計原則

### 1. 1シート1役割

記録・実験・知識・表示を混在させない。

### 2. Position Lab と リファレンスを分離する

`Position Lab` は研究ノート。  
`リファレンス` は試合で使う確定知識。

### 3. 知識は段階的に昇格する

```text
観察
↓
実験
↓
再現
↓
リファレンス
↓
試合
```

### 4. JSONを共通仕様とする

ChatGPT・GAS・Spreadsheet・将来Flutterは同一データモデルを共有する。

### 5. 現行Spreadsheetとの互換性を優先する

現時点では、理想設計よりも既存Spreadsheetの実態と安定運用を優先する。

## 非目標

- Reviewシート新設
- Reference候補の自動正式化
- 入力規則の破壊
- 数式列の上書き
- Flutter化
