# System Overview v2.0

## 目的

本システムは、ビリヤードの試合・練習・検証・プロ動画から得た情報を蓄積し、
試合で自然に再現できる判断力へ変換することを目的とした競技力向上システムである。

単なる練習記録ではなく、

**「観察 → 実験 → 知識化 → 試合で活用」**

のサイクルを継続し、競技力を向上させる。

# シート間の役割

| シート | 役割 |
|---|---|
| 練習ログ | 毎日の練習内容を記録する |
| ミスログ | 試合・実戦で外した配置や判断を記録する |
| Position Lab | 90度分離を基準にした変化量を実験・検証する |
| Reference | 検証済みの確定知識を蓄積する |
| Pro Reference | プロ動画から得た知識を管理する |
| 課題管理 | 継続して改善する課題を管理する |
| Today's Focus | 次回練習で意識する内容を表示する |

# 設計原則

### 1. 1シート1役割

記録・実験・知識・表示を混在させない。

---

### 2. Position Lab と Reference を分離する

Position Lab は研究ノート。

Reference は試合で使う確定知識。

---

### 3. 知識は段階的に昇格する

```text
観察

↓

実験

↓

再現

↓

Reference

↓

試合
```

---

### 4. JSONを共通仕様とする

ChatGPT・GAS・Spreadsheet・Flutterは同一データモデルを共有する。

---

### 5. 試合で自然に判断できる知識を増やす

本システムの目的は記録ではない。

試合中に迷わず判断できる知識を育てることを最終目的とする。
